5.
a. CSS Grid é um sistema de layout bidimensional que permite 
organizar elementos em linhas e colunas, oferecendo maior 
controle sobre o posicionamento.

b. Bootstrap é uma framework CSS que facilita o design de 
layouts responsivos através de classes pré-definidas e componentes 
prontos.

c. CSS Grid oferece maior flexibilidade para layouts personalizados,
enquanto o Bootstrap é mais rápido para layouts padrões, com uma 
abordagem baseada em classes fixas.
